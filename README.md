# Arduino Based Voting Machine

## Components Used
- Arduino UNO
- Push Buttons
- 16x2 LCD Display
- Buzzer
- PCB Board

## Description
This project implements a simple electronic voting machine using Arduino.
Each button represents a candidate. When a vote is cast, a buzzer sounds
and the LCD confirms the vote. A result button displays total votes.

## Applications
- Educational projects
- Mini project for ECE
- Voting system demo
